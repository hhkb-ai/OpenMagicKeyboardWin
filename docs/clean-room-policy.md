# Clean-room Policy

This project is an independent open-source implementation for Apple Magic Keyboard support on Windows.

## We can use

- Publicly observable behavior.
- User-generated HID logs.
- Microsoft's public Windows driver documentation.
- Open-source projects as references, while respecting their licenses.

## We must not use

- Decompiled commercial application code.
- Extracted commercial drivers.
- Proprietary icons, names, UI layouts, or paid-license logic.
- Binary patching or activation bypass code.

## License caution

Some relevant open-source keyboard projects use GPL licenses. If code is copied from them, this project may need to adopt GPL-compatible licensing. To keep the project MIT for now, use those projects as architectural references only and reimplement the code independently.
