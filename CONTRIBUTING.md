# Contributing

## Clean-room contribution rule

Do not submit code, binaries, UI assets, icons, text, or driver files copied from proprietary commercial software.

Allowed:

- Independent implementation.
- Public documentation.
- Original HID logs generated by your own keyboard.
- Ideas derived from observable behavior.
- References to open-source projects with compatible license handling.

Not allowed:

- Decompiled commercial binaries.
- Extracted commercial drivers.
- Copied proprietary UI or icons.
- License bypass logic.
- Code copied from GPL projects unless this repository's licensing strategy is changed accordingly.

## HID logs

When submitting HID reports, remove personal device names if needed. Do not include typed text. Use the logger's structured export.
